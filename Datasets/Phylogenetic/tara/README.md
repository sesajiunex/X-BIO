## Description

Tara Oceans data, Sunagawa et al. (2015).| This DNA data set consists of a RT with 3748 taxa, and a 16S rDNA reference MSA with 3374 sites. The QS are metagenomic reads, filtered to the 16S rDNA locus. From this dataset, we sub-sampled 10;000 QS.
