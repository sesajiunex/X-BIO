## Description

BV data, Srinivasan et al. (2012).| This DNA data set consists of a RT with 797 taxa, a 16S rDNA reference MSA with 2;763 sites, and 15;060 QSs. We refer to this as the bv data set.

## References

Bacterial communities in women with bacterial vaginosis: high resolution phylogenetic analyses reveal relationships of microbiota to clinical criteria. PloS one, 7(6), p.e37818.
